# **App Name**: VoltaView Auth

## Core Features:

- User Authentication: Secure user login and signup using Shadcn UI components.
- Responsive Design: Ensure the application is responsive and adapts to different screen sizes.
- Admin Authentication: Protect admin routes by checking a key stored in a .env file.  The key value must be 1250solaradmin to succeed in admin authentication

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), providing a professional, trustworthy, corporate feel.
- Background color: Very dark grey (#121212) providing contrast and modern look
- Accent color: Bright orange (#FF9100), drawing attention to important CTAs.
- Body and headline font: 'Inter', a sans-serif font for clear and accessible readability across the platform.
- Clean and modern layout, ensuring ease of use and visual appeal across all devices.
